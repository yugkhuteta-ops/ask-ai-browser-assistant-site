# Ask AI Browser Assistant — Landing Page

Modern, premium Next.js landing page for the **Ask AI Browser Assistant** browser extension.

## Features
- Next.js 15 + App Router
- TypeScript
- Tailwind CSS
- SEO (metadata, OpenGraph, Twitter cards)
- Static-friendly rendering

## Setup
```bash
cd "extension-ideas-agent/vercel sites"
npm install
npm run dev
```

## Build
```bash
npm run build
npm run start
```

## Vercel Deployment
1. Push this project to GitHub.
2. Create a Vercel project.
3. Set the root to `extension-ideas-agent/vercel sites`.
4. Build command: `npm run build`
5. Output is default for Next.js.
